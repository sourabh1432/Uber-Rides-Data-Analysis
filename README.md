# 🚖 Uber Rides Data Analysis (Power BI)

## 📊 Project Overview
This project analyzes Uber ride data to identify trends, peak hours, and ride patterns using Power BI.

## 🔧 Tools Used
- Power BI (Power Query, DAX)
- Excel / CSV

## 📌 Key Insights
- Identified peak ride hours during morning and evening
- Analyzed high-demand locations
- Observed weekday vs weekend ride trends
